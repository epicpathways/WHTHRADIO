How to Send Event Announcements

We happily broadcast Christian or community-related event announcements. Please follow the guidelines closely in order to have your announcement broadcast.
Guidelines
The event must fit within the station's Christian values.
  1. Timing: Submit the announcement early! One month before the event is best. Allow at least a week for production time.
  2. Write: Write the announcement exactly as you want it read and time it to fit 30 or 60 seconds. (Please do not send brochures, flyers, posters, etc.)
  3. Include: Event name, description, date, time, location, cost, phone number for more information. (Example: The Christian Singers, a world-famous a capella group, are coming to Park Rapids on July 7 at 7 p.m. at the Park Rapids Church on 100 Main Street. Admission is free! Call 218-111-0000 for more information. Learn more about the Christian Singers at www.christiansingers.org.)
  4. Music: For a musical event, if possible include an mp3 file of one of the songs to run as background.
  5. Email: Send the written announcement and mp3 file (if applicable) using this link. Email.
The station reserves the right to edit the announcement for time or content.
We will run every request possible, but cannot run all requests and cannot respond to every request. Thank you for understanding.
